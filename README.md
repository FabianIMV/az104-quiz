# ☁️ AZ-104 Quiz

A self-study quiz app for the Microsoft Azure Administrator (AZ-104) certification exam.

**Play it live:** https://fabianimv.github.io/az104-quiz/

## Features

- 70 questions across all 5 AZ-104 exam domains
- Shuffled question order every session
- Instant feedback with explanations after each answer
- Review round: incorrect answers come back for a second pass
- Domain breakdown score on the results screen
- Clean, mobile-friendly UI

## Domains covered

1. **Identities & Governance** — Azure AD roles, RBAC, policies, subscriptions
2. **Storage** — Blob, Files, redundancy, access tiers, SAS tokens
3. **Compute** — VMs, App Service, containers, AKS, availability sets
4. **Networking** — VNets, NSGs, load balancers, VPN, ExpressRoute
5. **Monitoring & Maintenance** — Azure Monitor, Log Analytics, Backup, Site Recovery
